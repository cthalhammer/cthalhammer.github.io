## Course Overview

This course provides an introduction to numerical methods specifically tailored to physics and electrical engineering students.The curriculum covers essential numerical techniques in analysis (root finding, integration) and ordinary differential equations. The emphasis is on developing practical skills in applying numerical methods to solve physics problems using Python programming.

Students gain hands-on experience with scientific Python libraries including numpy, scipy, sympy, and matplotlib, learning to implement and apply fundamental algorithms for solving basic numerical problems in physics.

## Core Topics

### Linear and Nonlinear Least Squares Fitting
- Linear regression and parameter estimation
- Nonlinear curve fitting techniques
- Error analysis and uncertainty propagation
- Applications to experimental data analysis

### Nonlinear Equations
- **Scalar equations**: Newton-Raphson method, bisection, secant method
- **Systems of equations**: Multidimensional Newton's method, fixed-point iteration
- **Physics applications**: Finding equilibrium points, solving transcendental equations

### Numerical Integration
- **Quadrature rules**: Trapezoidal rule, Simpson's rule, Gaussian quadrature
- **Adaptive integration**: Error estimation and mesh refinement
- **Monte Carlo methods**: Random sampling and importance sampling
- **Applications**: Computing physical quantities, evaluating definite integrals

### Initial Value Problems for Ordinary Differential Equations
- **Euler methods**: Forward and backward Euler, improved Euler
- **Runge-Kutta methods**: Classical RK4, adaptive step-size control
- **Multistep methods**: Adams-Bashforth, Adams-Moulton
- **Physics applications**: Particle dynamics, oscillators, population models
